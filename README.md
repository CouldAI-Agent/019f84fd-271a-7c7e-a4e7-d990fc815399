# NexaNews WordPress Theme

A premium, fast, and responsive WordPress news and magazine theme.

## Features
- **Responsive Design**: Works perfectly on mobile, tablet, and desktop devices.
- **Custom Homepage**: Built-in template for a feature-rich homepage layout with featured news and latest posts.
- **Clean Code**: Built following WordPress Coding Standards without heavy page builders or frameworks like Bootstrap.
- **Widgetized Areas**: Multiple sidebar and footer widget areas for easy customization.
- **SEO Optimized**: Clean HTML5 and CSS3 structure.

## Setup Instructions

1. Ensure you have a working WordPress installation.
2. Copy the `nexanews` folder to your `/wp-content/themes/` directory.
3. Log in to your WordPress Admin dashboard.
4. Go to **Appearance > Themes**.
5. Find **NexaNews** and click **Activate**.

## Theme Structure
The theme uses a straightforward WordPress structure:
- `style.css` - Main stylesheet and theme declaration.
- `functions.php` - Theme setup, enqueues, menu registration, and widget areas.
- `header.php` - Site header.
- `footer.php` - Site footer.
- `index.php`, `front-page.php`, `single.php`, `page.php` - Core templates.
- `sidebar.php` - Main sidebar area.
